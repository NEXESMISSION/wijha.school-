Le parcours

Ce que tu fais, étape par étape

1 projet simple, en 1 session live de 2h : du premier clic à ton site en ligne — et les outils que tu vas utiliser.

1
Inspiration · IA

Copier un design

Tu trouves un site que t'aimes, et t'apprends à le recréer à ta sauce. Pas besoin d'imagination — on part de ce qui marche déjà.

  • Repérer les bons sites de référence
  • Capturer le style : couleurs, sections, ambiance
  • Bien décrire le résultat voulu à l'IA
2
Cursor · Claude Code

Construire avec l'IA

Le cœur du métier : tu parles à l'IA en français, elle écrit le vrai code de ton site. C'est toi qui pilotes, l'IA exécute.

  • Installer Cursor + Claude Code (guidé pas à pas)
  • Générer ton site en le décrivant
  • Tout modifier en parlant, sans coder à la main
3
Multi-pages

Plusieurs pages

Un vrai site, pas une seule page : accueil, à propos, services, contact… reliés par un menu propre et clair.

  • Créer plusieurs pages reliées
  • Un menu de navigation propre
  • Une structure claire pour le visiteur
4
Animations · Vitesse

Designer & animer

On rend ton site beau ET vivant : des animations fluides, parfait sur téléphone, et rapide à charger.

  • Animations & effets au défilement
  • 100% responsive (mobile-first)
  • Optimisation vitesse & images
5
Google Sheets · Telegram

Formulaire qui marche

Ton site capte de vrais messages : chaque formulaire rempli arrive dans un Google Sheet, et te prévient direct sur Telegram.

  • Un formulaire de contact connecté
  • Les réponses rangées dans Google Sheets
  • Une notification instantanée sur Telegram
6
GitHub · Vercel · Domaine

Mettre en ligne

On publie ton site pour le monde entier, avec ton propre nom de domaine — du code jusqu'au lien public.

  • Sauvegarder ton code sur GitHub
  • Déployer sur Vercel / Cloudflare
  • Connecter ton domaine (Hostinger)
